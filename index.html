Tilgang til vSphere

For å opprette en VM i VirtualBox, følg disse stegene:

  1. Naviger til vSphere web-klienten i nettleseren din: https://10.10.1.20/ui

    2. For å opprette en VM i VirtualBox, følg disse stegene:

  2. Høyreklikk på verten eller clusteret
  3. Velg "Ny Virtuell Maskin"
  4. Velg "Opprett en ny virtuell maskin"
  5. Velg navn og plassering for VM-en

    6. Konfigurer VM-innstillinger

  6. CPU: Minimum 2 kjerner
  7. Minne: Minimum 4GB
  8. Lagring: Minimum 20GB
  9. Nettverk: Konfigurer nettverksadapter
    10. Merk: Dette er minimumsspesifikasjoner. Juster basert på dine behov.

Start og Installere Ubuntu

For å starte Ubuntu på VM, følg disse stegene:

  1. Trykk på "Start"
  2. Velg ISO-fil
  3. Trykk "Start"
  4. Velg språk og trykk "Install Ubuntu"
  5. Velg "Normal Installation"
  6. Velg "Download updates while installing Ubuntu" og "Install third-party software"
  7. Velg "Erase disk and install Ubuntu" og trykk "Install Now"
  8. Velg tidssone og tastaturoppsett
  9. Velg brukernavn og passord
  10. Trykk "Restart Now"

Installere Ubuntu Webserver Nginx

For å installere Nginx, følg disse stegene:

  1. Åpne terminal
  2. Skriv "sudo apt update"
  3. Skriv "sudo apt install nginx"
  4. Skriv "sudo ufw allow 'Nginx HTTP'"
  5. Skriv "sudo ufw status"
  6. Skriv "systemctl status nginx"
  7. Åpne nettleser og skriv inn IP-adressen til VM

Lag Nettsiden

For å lage en nettside, følg disse stegene:

  1. Åpne terminal
  2. Skriv "sudo nano /var/www/html/index.html"
  3. Skriv inn HTML-kode
  4. Trykk "Ctrl + X" og "Y" for å lagre
  5. Åpne nettleser og skriv inn IP-adressen til VM

Deploy Fra Git

For å deploye nettsiden fra Git, følg disse stegene:

  1. Åpne terminal
  2. Skriv "sudo apt install git"
  3. Skriv "cd /var/www/html"
  4. Skriv "sudo git clone URL"
  5. Åpne nettleser og skriv inn IP-adressen til VM

Restart Nginx

For å restarte Nginx, følg disse stegene:

  1. Åpne terminal
  2. Skriv "sudo systemctl restart nginx"

Tips

Her er noen tips for å bruke Ubuntu:

  • For å oppdatere Ubuntu, skriv "sudo apt update" og "sudo apt upgrade"
  • For å installere programmer, skriv "sudo apt install PROGRAM"
  • For å slette filer, skriv "rm FILE"
  • For å restarte Ubuntu, skriv "sudo reboot"
